# v1.0.0

Traffic Monitor 首个可正式发布的稳定版本。

## Highlights

- 重做首页监控面板，单屏整合趋势、汇总指标、排行、目标主机和连接明细。
- 将前端静态资源拆分为 `index.html`、`styles.css`、`app.js`，后续维护和迭代更直接。
- 新增 Mihomo 连接设置面板，支持在页面内配置和更新 Controller 地址与 Secret。
- 优化长文本场景，代理名、主机名、连接明细标题支持截断或两行换行，避免面板挤压错位。
- 收紧页面纵向高度，趋势区、汇总卡片、排行区、主机区、明细区都做了更适合长期挂机观察的紧凑布局。

## Included In This Release

- 顶部时间范围和维度切换交互整理，保留手动刷新与自定义时间范围。
- 首页增加 GitHub 与 MIT License 入口。
- 下方三栏面板统一做高度约束与内部滚动，长列表不再无限拉长整页。
- 汇总卡片重新排布为两行两列，`上传` 与 `下载` 同行展示。
- 中间栏和右侧明细标题保留完整上下文，同时限制为最多两行显示。
- 多处前端样式细节修正，包括代理排行长名称、搜索框宽度、表头密度和标题换行策略。

## Notes

- 该项目当前定位为局域网内使用的流量监控工具。
- 发布前已完成 `go test ./...` 与 `go vet ./...` 验证。
